## 🔧 Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
/* Update gradient colors */
background: linear-gradient(135deg, #1e3c72, #4dff88);
```

### Adding New Courses
1. Add course data to `courseDetails` object in `script.js`
2. Add course card HTML to `index.html`
3. Update course grid styling if needed

### WhatsApp Integration
Update the phone number in `script.js`:
```javascript
function redirectToWhatsApp() {
    window.open('https://wa.me/YOUR_PHONE_NUMBER', '_blank');
}
```
